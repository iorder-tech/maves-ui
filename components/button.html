Button #

The m-button component replaces the standard html button.

Examples #

Primary #


<template>
	<m-button>Primary</m-button>
</template>

Ghost #


<template>
	<m-button type="ghost">Ghost</m-button>
</template>

Inactive Button #


<template>
	<m-button type="ghost">Ghost</m-button>
</template>

Text with Icon #


<template>
	<m-button iconName="text">Text with Icon</m-button>
</template>

Icon #


<template>
	<m-button iconName="text" icon />
</template>

Props #

NameTypeDefaultDescription

iconbooleanfalseHide text in the button
iconNamestringChoose icon
maxWidthstringSet maxWidth to button
