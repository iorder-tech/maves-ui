Button #

The m-button component replaces the standard html button.

Examples #

Primary #


vue
<template>
	<m-button>Primary</m-button>
</template>

Ghost #


vue
<template>
	<m-button type="ghost">Ghost</m-button>
</template>

Inactive Button #


vue
<template>
	<m-button type="ghost">Ghost</m-button>
</template>

Text with Icon #


vue
<template>
	<m-button iconName="text">Text with Icon</m-button>
</template>

Icon #


vue
<template>
	<m-button iconName="text" icon />
</template>
